# 5️⃣ Conclusion
### 기대효과
- Simple : 손의 제스쳐만으로 마우스 제어
- Comfortable : 자유롭게 움직일 수 있는 편안함
- Usability : 다양한 분야에 활용 가능
- Health : 마우스 증후군 예방

- VR에 적용한 현실 같은 가상 공간
- 근전도 센서를 활용하여 의수에 적용
